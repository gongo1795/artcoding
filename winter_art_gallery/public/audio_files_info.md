# 오디오 파일 안내

이 디렉토리에는 다음 오디오 파일들이 필요합니다:

## background.mp3
- 갤러리 배경음악 파일
- 차가운 겨울 분위기에 맞는 앰비언트 음악 권장
- 루프 재생을 위한 매끄러운 시작/끝 구성 필요

## Explain.mp3
- 큐레이터 작품 설명 나레이션 파일
- curator_narration.txt 내용을 음성으로 녹음한 파일
- 명확하고 차분한 톤의 나레이션 권장

## 파일 추가 방법
1. 해당 오디오 파일들을 이 public 디렉토리에 업로드
2. 파일명이 정확히 일치하는지 확인
3. 웹 브라우저에서 지원하는 형식(MP3, WAV, OGG) 사용 권장

현재는 플레이스홀더 상태이므로 실제 오디오 파일을 추가하면 자동으로 작동합니다.
